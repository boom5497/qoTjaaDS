# 项目简介

## 核心功能
- 基于 YOLOv8 的遥感目标检测系统
- 包含完整源码、20865 张标注数据集（涵盖 12 类目标）、训练好的模型及训练结果
- 提供项目运行教程，支持 Python 3.9 + OpenCV-Python + PyQt5 + Ultralytics + Torch1.9 环境
- 支持多种输入源检测：单张图片、图片文件夹、视频文件、本地摄像头
- 界面功能：实时显示目标位置、目标总数、置信度等信息
- 支持自定义置信度和 IOU 阈值设置
- 提供模型选择功能
- 支持批量检测并查看所有结果
- 支持检测结果保存
- 可选择是否显示标签和原图

## 适用场景
- 遥感图像分析
- 城市规划与管理
- 环境监测与研究
- 交通流量分析
- 基础设施检查
- 土地利用分类

## 技术特点
- 采用先进的 YOLOv8 目标检测算法
- 提供 PyQt5 图形用户界面
- 支持多种输入格式和实时检测
- 内置批量处理功能
- 模块化设计，易于扩展和修改
